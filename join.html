参加方法

マイ謎会に参加するには、以下のステップを踏んでください。

1. 会員登録をする

マイ謎会は会員登録が必要となっております。
以下のリンクから申し込みお願いいたします。

※運営が手作業で入力を行っているため、お申し込み完了から登録完了まで数日かかる可能性があります。

会員登録申込

2. Discordに参加する

マイ謎会はDiscordで運営されています。以下のリンクから参加してください。

Discordに参加する

3. Minecraftの準備

参加にはMinecraft Java Edition または
Minecraft Bedrock Editionが必要です。必要に応じてインストールしてください。

4. 開催情報をチェック

イベントの開催情報はDiscordまたは公式サイトでお知らせします。

5. イベントに参加！

イベントに参加しましょう！自分で作った謎解きを紹介するのもOKです！
